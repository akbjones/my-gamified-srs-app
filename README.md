## SRS Logic (Subheading)
| User Grade | Ease Factor ($E$) Change | Interval ($I$) Formula |
| :--- | :--- | :--- |
| **Forgot** | -0.20 | Reset to 0 |
| **Hard** | -0.15 | $I = I \times 1.2$ |
| **Good** | No change | $I = I \times E$ |
| **Easy** | +0.15 | $I = I \times E \times 1.3$ |
* Rule 1: XP starts at 0 (Bullet point)
